# Company Bankruptcy Classification

This project applies machine learning classification algorithms to predict whether a company is **bankrupt** or **financially healthy** based on a dataset containing financial indicators.

The dataset is **imbalanced**, with many more healthy companies than bankrupt ones.

## Models Used

The following algorithms are implemented:
- Decision Tree Classifier
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

## Evaluation

Model performance is evaluated using **confusion matrices**, which show the distribution of correct and incorrect predictions for the two classes (bankrupt vs healthy).
